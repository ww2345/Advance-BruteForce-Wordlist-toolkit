# 🔥 Wordlist Toolkit Pro

**Wordlist Toolkit Pro** is an interactive Bash-based CLI tool designed to generate structured wordlists using multiple input sources. It provides a menu-driven interface with powerful features for combining, merging, and generating datasets efficiently.

---

## ✨ Features

* 📁 Combine multiple files (full permutations / Cartesian product)
* ⚡ Fast and memory-efficient processing
* 🎛️ Interactive menu-driven CLI
* ⌨️ Tab auto-completion support
* 🔀 Custom separator & reverse mode
* 🔢 Numeric wordlist generator
* 📦 Merge multiple wordlists
* 📘 Built-in usage guide with examples
* 🔐 Basic integrity check (tamper detection)
* 🎨 Clean and beginner-friendly interface

---

## 🚀 Installation

```bash
git clone https://github.com/ww2345/Advance-BruteForce-Wordlist-toolkit
cd WordlistToolkit
chmod +x wordlistGenraterToolkit.sh
```

---

## ▶️ Usage

```bash
./wordlist_toolkit.sh
```

Follow the interactive menu to:

1. Combine multiple files
2. Merge wordlists
3. Generate numeric sequences
4. View usage guide

---

## 📌 Example

**Input:**

file1.txt

```
anup
raj
```

file2.txt

```
123
456
```

**Output:**

```
anup123
anup456
raj123
raj456
```

👉 The tool combines **every entry from one file with all entries from others**, not just one-to-one matching.
